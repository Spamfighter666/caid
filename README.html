Method and Publication Notes

This Phase 1 website publishes a sanitized public export of Sheet1 from the CAID master workbook.

The public site intentionally omits internal analysis fields, network intelligence layers, monitoring notes, and other analyst-facing metadata. Records are presented for browsing, filtering, and record-level review.

Public fields

Date, city, province, target, target type, incident type, public summary, severity, arrests, charges, and source.

Private fields withheld

Internal monitoring notes, network mappings, actor analysis, investigative flags, and other analyst-use columns.

Deployment model

Static GitHub Pages site driven by a JSON export derived from the private master workbook.
